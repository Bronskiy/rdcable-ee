About This Page

The "Entry Details" page displays info for the selected channel entry.

It makes use of the following tags:

  • EE Channel:Entries to display channel entry data.
  • Tag:Tags to display the tags currently assigned to the entry.
  • Tag:Related_Entries to display a list of entries that share some of the same tag(s).
  • Tag:Cloud to display a list of most popular tags on the right sidebar.
