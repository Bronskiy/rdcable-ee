About This Page

The "Tag Entries" page displays a list of entries that have been assigned the tag that has been searched on.

It makes use of the following tags:

  • EE Channel:Entries to display a list of channel entries.
  • Tag:Tag_Name to display a nicer version of the tag name being searched on.
  • Tag:Tags to display the tags currently assigned to each entry.
  • Tag:Cloud to display a list of most popular tags on the right sidebar.
